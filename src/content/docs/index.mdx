# Engenharia de software

Não basta apenas criar soluções eficientes e eficazes para problemas reais. A
engenharia de *software* aborda como criar sistemas escaláveis, seguros e
estáveis garantindo a qualidade do código.

## Tópicos

* Arquitetura de sistemas
* Arquitetura e engenharia de dados
* Qualidade de código
* DevOps
* Testes e validação
* Segurança
